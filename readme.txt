== Description ==

Invite purchasers to make a donation to your organization when they make their purchase.

New or updated translations are always appreciated. The translation files are included in the download. 

== Installation ==

1. Upload the `/mt-donations/` directory into your WordPress plugins directory.

2. Activate the plugin on your WordPress plugins page
  
3. Add your license key in the My Tickets > Settings screen.
